## 📱 Overview

**CricketScoreCard** is a modern Android application designed for scoring cricket matches with professional-grade statistics and scorecards.

Whether you're managing a local tournament, club match, corporate event, or weekend game, CricketScoreCard provides an intuitive experience for recording every ball, tracking player performance, and generating detailed match reports.

---

## ✨ Features

### 🏏 Match Creation

* Create custom cricket matches
* Configure teams and player count
* Flexible overs format
* Virtual toss system
* Resume ongoing matches

### ⚡ Live Ball-by-Ball Scoring

* Runs and extras tracking
* Wide, No Ball, Bye, Leg Bye support
* Wicket management
* Strike rotation
* Undo last delivery

### 📊 Professional Scorecards

* Complete batting statistics
* Detailed bowling figures
* Extras breakdown
* Innings summaries
* Match result information

### 👤 Player Statistics

* Career runs
* Career wickets
* Matches played
* Batting records
* Bowling records

### 🏆 Leaderboards

* Highest run scorers
* Leading wicket takers
* Ranked player statistics
* Performance tracking

### 👥 Team Analytics

* Matches played
* Wins and losses
* Team records
* Squad information

### 📄 PDF Export

* Generate professional scorecards
* Download match reports
* Share completed matches

### 🎨 Modern User Experience

* Material Design 3 UI
* Dark theme interface
* Smooth animations
* Responsive layouts
* Optimized performance
